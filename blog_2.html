My 100 Days of code challenge

November 22, 2020

Finally I've completed this challenge it took me more than 100 days. In between this challenge, I had some college assignments and also gave my final semester exams.


#100DaysOfCode

earlier I was using simple HTML like div, tables and basic CSS styles like px, floats to create web pages.
I've never used flexbox, semantic elements, fonts, relative units etc. I've been using sublime then at the start of this year a video popped up in YT feed.


It was a Web Development Roadmap 2020 by @codeSTACKr . I watched the video and subscribed to his channel. He had less than 8k subs then slowly started using Visual Studio Code and it was a new experience. I learned Emmet, box models, queries, relative units etc from his videos.

but I was not consistent due to college assignments and exams. I remember in march he dropped a video about free resources to learn web development. In this video, He suggested @freeCodeCamp 's website and their YT channel.

I joined fCC's responsive web design course and also subscribed to their YT. 2 week later due to covid lockdown declared, for me it was perfect time cause I got enough time to learn and practice. I learned HTML and CSS if I stuck at any point or I wanted more insight to a topic.

I searched on YT and learned all. @codeSTACKr, @Colt_Steele, @john_smilga, @freeCodeCamp, @KevinJPowell thanks to all.

In b/w this I followed fCC and @codeSTACKr on twitter. Earlier, I used to see memes and stuff on twitter after following these two accounts I got to know abt the @_100DaysOfCode challenge. I accepted the challenge after learning HTML and CSS, I created some web pages

but I want to show these to other people. so I learned to use @github but using GUI (@GitHubDesktop) then learned CLI (@git).

The next I wanted to learn was JavaScript. which was a challenge because I didn't know where to start @codeSTACKr has great tutorials but I need to learn more. fCC has a free JavaScript Algo and DS course I enrolled into and started learning. Then found @thenetninjauk YT channel.

He has a modern JavaScript course on Udemy, In this course he has everything I wanted to learn. Currently learning to use babel and webpack with JS.

Now, I can create web pages which use modern tech, I know to use VScode, git, GitHub, tailwind, chromeDevTools also I can read documentation and learn new tech. thanks to @codeSTACKr for sharing awesome resources.

My next step would be to learn, react and get a job.
